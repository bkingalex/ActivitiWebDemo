# ActivitiWebDemo

### 技术选型
>SSM + MySQL5.6 + Activiti 5.22
<br>

### 功能点
#### 1.审批流程管
    1.1 审批流程列表
    1.2 可删除定义的审批流程
    1.3 可查看部署的流程图
    1.4 部署流程定义文档   


#### 2.申请模板管理
    2.1 添加模板
    2.2 修改模板
    2.3 删除模板
    2.4 下载模板
    
#### 3.起草申请
    3.1 申请列表
    3.2 上传申请单
    
#### 4.待我审批
    4.1 带我审批 列表
    4.2 审批处理，填写审批意见。是否同意
    4.3 查看审批记录
    
#### 5.我的申请查询
    5.1 我的申请 列表
    5.2 查看审批记录
